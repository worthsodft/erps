## 部署体验
### 快速部署
文件解压完成后，在终端项目根目录运行以下命令即可体验。
  ```shell
php think run --host 127.0.0.1
  ```

### 部署步骤
1.创建数据库
2.修改数据库参数
3.终端运行项目

## 标准版和专业版的区别
1. 标准版包含了基础的进、销、存管理功能，能满足大部分电商业态、业务场景，适合大多数电商使用。
2. 专业版包含了标准版的所有功能，还包含了一些个性化的功能，如：实物卡管理、分享推广及海报管理等。
